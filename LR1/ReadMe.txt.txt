1 Запустить Main.java

2 Ввести любое число от 0 до 3 

3 Если ввели 0 - Программа завершит свою работау

4 Если ввели 1 - Программа перейдет в режим создания файла

4.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите C:/
4.2 Программа запросит имя файла. Для примера: введите test.txt
4.3 Программа создаст файл в указанном месте и перейдет в режим ожидания следующей команды

5. Если ввели 2 - Программа перейдет в режим создания файла c данными в проекте FileManager
5.1 Программа создаст файл с данными и перейдет в режим ожидания следующей команды

6. Если ввели 3 - Программа перейдет в режим удаления файла

6.1 Программа запросит путь к файлу (обязательно - в конце пути необходимо указывать символ "/"). Для примера: введите C:/
6.2 Программа запросит имя файла. Для примера: введите test.txt (этот файл должен существовать, иначе программа выведет ошибку о том, что такого файла нет)
6.3 Программа удалит файл и перейдет в режим ожидания следующей команды
